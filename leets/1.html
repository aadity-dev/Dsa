Concatenate Non-Zero Digits and Multiply by Sum II

Sparse Table (disjoint power-of-two decomposition) · Hard

For each query [l,r], concatenate the non-zero digits of s[l..r] into x (0 if none), let sum be the digit sum of x, and return x × sum mod 1e9+7. A sparse table stores, for every block of length 2^k, its concatenated value/digit-count/digit-sum so any range can be answered by merging O(log n) disjoint blocks.
